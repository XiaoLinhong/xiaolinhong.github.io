前言

hexo写文章时，需要了解三样工具，包括markdown、标签插件和MathJax，其中标签插件是对markdown语法的扩展,MathJax则是用来编写数学公式。

markdown

markdown 一种简单、易读的纯文本标记语言，纯文本文件本身没有格式的概念，而标记语言为纯文本文件的渲染提供可能。

文本编译涉及两件事：

  • 专注内容的编写（普通的文本文件）；
  • 内容格式的编排（复杂的word和HTML）；

markdown的出现是帮助我们在专注内容的编写的同时，简单地标记一下格式。

  • 标记异常简单；
  • 未被渲染之前，亦可阅读；
  • 容易转化成其它存储格式；

markdown的语法主要包括标题、列表、引用、字体（斜体或者粗体）、链接、代码块等。

插入特殊块

标签插件的存在为markdown提供了额外的语法特性，标签插件和多数模板引擎的语法相似。 当然标签插件一些功能与markdown重复，建议使用标签插件的语法。

  • blockquote # 引用
  • codeblock # 代码块
  • img # 图片
  • link # 链接
